Discription:

This 26th century ship is featured in the Ships of the Line Calender.  Only one view was shown ever of this ship, so I guessed on the rest.  There could be so many variations of this ship.

Requirements:

foundation (for any mod to work)

NanoFx 2.0 beta (for blinkers) 


__________________________________
Installation Instructions:

1. After installing BC v1.1, and the foundation mod, NanoFx 2.0 beta, unzip this mod into a temporary directory.

2. Copy files (data, scripts, and sfx) into your Bridge Commander folder. Click "Yes" to overwrite.  

3. Once ingame, you should see a ship selection called U.S.S. Enterprise J.  Select it and enjoy!
