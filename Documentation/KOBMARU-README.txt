README Kobayashi Maru:
==========
Kobayashi Maru (KM) - Version 2011.10

This is a Bridge Commander Mod pack.
Please read our Manual in the doc/ -folder.

If you find a bug, please report it here:
http://bckobayashimaru.de


Changelog:
<---------->
For changes since KM 1.0 see our wiki: http://bckobayashimaru.de/wiki/index.php/Changes_KM1.1


Install:
<---------->
Kobayashi Maru has to be installed on a clean BC 1.1 installation:
1. Install Bridge Commander.
On Windows Vista and Windows 7 don't even think of installing BC into the Program Directory (aka c:\Program files).
Install it somewhere else like c:\Games.
2. Install the 1.1 patch
3. Install KM2011.10, make sure all stock BC 1.1 files gets overwritten.

thats all
If something does not work, please take a look at KOBMARU-FAQ.txt in the doc/
Folder first before contacting us or a Forum.


Update:
<---------->
Upgrading will only work on clean, working FBCMP 0.8 or older KM installations.
You need to run the km_clean.bat after the upgrade


Bridge Commander Universal Tool notes:
<---------->
This KM version comes with an updated versions.ini for BCUT. Please update your local BCUT by copying
tools/BCUT/kobmaru/versions.ini to kobmaru/versions.ini of your BCUT installation.
