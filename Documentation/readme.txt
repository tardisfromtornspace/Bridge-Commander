Description :

This is the Constellation class USS Stargazer. Model was ported from SFC2 and completly retextured and rehardpointed. Hardpoints are balanced to match most of TMP ships including latest Constitution pack in battle, phasers fires short bursts, but beware this ship has only forward and side weapons coverage do don't let Your enemy get on Your back ;) This mod will not overwrite any Constellation class.

!!! IMPORTANT !!!
this is not final HP so be patient if something is over or under powered or missed


Installation :

Copy and paste Data, Scripts and SFX folders to the Bridge Commander root directory. 
Don't use BCMP, all the work has been done for You.
