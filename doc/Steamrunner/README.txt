Steam Runner

Original Mesh / Textures by Dan Huling
Anything else by Kevo


--------------------------

This mod REQUIRES Dasher42’s Ship plug-in system look for it here:
http://dynamic3.gamespy.com/~bridgecommander/vbforums/showthread.php?s=&threadid=1348 

--------------------------

INSTALLATION:

Copy both directories to the root bridge commander directory.  
Edit Dasher42’s Plugins.py and add the following:

import Steamrunner

Save it and your done.



Enjoy!
