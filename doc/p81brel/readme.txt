[Installation]
*remember*: you must have foundation plug-in installed. download it at http://www.bcfiles.com
find the stock bop model folder Data/models/ships/BirdOfPrey and delete EVERYTHING in it. Then:
unzip this zip in a temporary folder and then copy data and scripts directories in you activision/bridge commander folder. if asked overwrite the old files and all files will be updated. 
Delete matching pyc files in scripts/ships and scripts/ships/hardpoints

if you can't get the ship ingame contact me at book@bcfiles.com
