README:
==========
Version 0.5

Launching Mines.
After clicking on prepare mine, Brex will need 40 seconds to finish it.
After that time you can launch it using the launching framework.
The mine will explode when a Target is coming in range of ~10km

Requirements:
<---------->
-QBautostart 0.6 or higher
-New Version of the Shuttle Launching Framework
 if you are unsure where to find the new Version, look here:
 http://dynamic3.gamespy.com/~bridgecommander/phpBB/viewtopic.php?t=16490
-NanoFX1 or 2 to get a nice explosion


Install:
<---------->
Just put the files into their right place...
If you have NanoFX2 beta installed, please install also the patch included
else exploding mines will crash the game with a chance of 20% - don't know why.


Bugs:
