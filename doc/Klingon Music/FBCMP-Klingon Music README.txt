README FBCMP:
==========
Fun Bridge Commander Multi Player Project - Klingon Music Add On

Playing as Klingon you will now hear Klingon Music from the game Klingon Academy.


Authors:
<---------->
Inon Zur, composer of the KA Music.
Awatenor, packager for KDF (Klingon Defence Mod for BC).

Packaged by the people of the German Star Trek Forum at http://StarTrek.webhop.info.


Requirements:
<---------->
FBCMP 0.8	...well this should work on every BC installation with NanoFX2.


Credits:
<---------->

Inon Zur and Awatenor for the Music
