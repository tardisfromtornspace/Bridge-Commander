Stock Asteroid Replacment Textures 1.0

These are Hi-Res Textures.
==============================WARNING===============================

These Textures will replace your stock asteroid textures. If you would like to keep you old one
please make a backup copy.

=====================Installation Instructions======================

Take the folder labled "data" and drop in you home BC directory. When asked to overight click
"yes"

===========================Requirements=============================

All you need is a fresh install of Bridge Commander.
