====Legends of Armada: Romulan Raptor=====

Credits:
Design: From Startrek Armada by Activision
Model/texture: Capt LC Amaral
Hardpointing/testing: Starforce2
Torpedo and sound by JLH

Armaments:
1  dorsal and ventral defensive beam disruptor
4 fore and aft pulse disruptor
1  foreward torpedo tube
Cloaking device and tractor beams.


Low res textures available in the appropriate folders. RaptorX stands for the Startrek X valdor color scheme. Each nif uses it's own texture, 1 texture per ship.  There are scripts for both so you can install them both at the same time, there is no difference in performance between the 2.
