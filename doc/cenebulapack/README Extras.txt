CE NEBULA CLASS EXTRA FILE PATCH V.1

This is an optional Patch for the CE Nebula Class, it contains canon .nif files complete with the square deflector, blank regs and alternate deflector textures.
This pack replaces the corrupted files that came with the original release.

NIF INSTALLATION.

1) Simply copy and paste the files into your data/models/ships/Nebula/high folder and overwrite existing files. Please remember to backup the original files before overwriting them.

2) delete any .vox files in your Nebula/high directory.
