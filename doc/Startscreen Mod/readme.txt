This is a mod that changes the BIK Video in the Sart-Screen!

If you start the game and scip the intros you will see the "New Game" Window.

Normaly there is the Starmap, but if you have correct installed it you will see some parts of Star Trek Voyager!

-------------
Requirements: Your resolution in the Game MUST be 1024x768 Pixel!  If dont, you will see the normal Screen!
-------------

Install:

Make a Backup of your original: \data\movies\IF_Starmap1024768

Just copy the file-order in the BC root Directory.

If you choose the file, copy it to: ?\Bridge Commander\data\movies\



Ready?

Now start the Game!
