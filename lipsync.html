Lip Syncing

Bridge Commander supports lip syncing for its characters. To use it, you need to generate a phoneme (.LIP) file for each sound file in the game. If a .LIP file is present in the same directory as a line a character is speaking, it will be used to lip sync the character. If there is no .LIP file, the game will use some random phonemes instead.

To produce .LIP files, do the following:

  • Download and install LISET, Microsoft's free linguistic sound analysis tool (part of the MS Agent developer tool set).
  • Load the sound file (.wav format) you wish to process into LISET. Enter the text that the line represents into the Text Representation field. Then, select the Generate Linguistic Info option from the Edit menu. LISET will then process the file, and generate phonemes for the text you entered.
  • When it's done, you should have a bunch of rectangular boxes in the waveform display, representing the phonemes generated. Save the file (a "linguistically enhanced wave file") to the same location as the source file.
  • If LISET was unsuccessful in producing phonemes for the wave file (no phoneme rectangles), you can have a couple of options:
    • Check to make sure that the text matches the line exactly.
    • Adjust the text to more closely match the pronunciation of the line. Possessives, words that run together, and punctuation are all things that might be causing trouble.
    • Record a line in the same meter as the spoken line, but enunciated more clearly. Some voices are understood better by LISET, for some reason.
    • Manually place phonemes in LISET. See the LISET documentation for instructions on how to do this.
  • Run Bridge Commander with the -TestMode option, to enable the console. (tilde key by default) Enter the following into the console: 
    App.CharacterAction_ConvertFolderLWVs(path)
    where path is the relative path of the directory of .LWV files to convert. (i.e. "sfx/Bridge/Crew/Admiral Liu") It does not recursively process directories, and it will not delete the .lwv files when it is done. It just generates the .LIP files needed for lip syncing.
  • After you have produced the .LIP files, you no longer need the .LWV files. You can delete them if you want.
