Power Saver Functions

We have discovered through the course of testing that many computers equipped with a power saver feature in their CMOS have problems running the game when this feature is turned on. This feature only activates after the computer is left idle for a certain period of time. To solve this problem, you should exit the game if you plan to be away from the computer for a long period of time. Another option is to turn this feature off in your system CMOS.
