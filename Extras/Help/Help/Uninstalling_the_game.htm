Uninstalling the Game

How to uninstall the game:

The game's Uninstall option allows the game to remove itself completely from your hard drive. 

From the Start button, go to Programs/Star Trek: Bridge Commander/Uninstall and follow the Uninstaller instructions.

NOTE: You should always uninstall the game with the Uninstaller provided.

If you remove the game from your hard drive by dragging your Star Trek®: Bridge Commander™ folder to the Recycle Bin, certain files may be left behind causing problems with a later reinstallation of the game. In order to completely remove all Star Trek: Bridge Commander files from your computer, complete  the following steps:

1. Make sure you have deleted the entire directory to which you installed Star Trek: Bridge Commander.

2. Remove Star Trek: Bridge Commander from the Start menu.

  1. From the Start button, go to Settings->Taskbar.
  2. Select the Start Menu Programs tab.
  3. Click Remove.
  4. Highlight and then delete the entire Star Trek: Bridge Commander entry.
