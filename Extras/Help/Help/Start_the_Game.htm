Starting the Game

There are three common ways to start Star Trek®: Bridge Commander™.

1. Use the Start Menu to run the game

  • Click the Start button on the task bar at the bottom of the screen
  • Select Programs
  • Select Star Trek Bridge Commander
  • Click on the Star Trek Bridge Commander shortcut

2. Insert the CD-ROM into your CD-ROM drive and click Play  when the title screen appears.

3. In My Computer double click on the CD-ROM drive icon (or right-click the CD-ROM drive icon and select Autoplay) and click Play when the title screen appears.
