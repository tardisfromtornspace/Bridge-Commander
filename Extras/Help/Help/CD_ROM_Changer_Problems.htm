CD-ROM Changer Problems

If you have a CD-ROM changer and experience problems launching the game, you need to play the game using the same CD-ROM drive from which you installed it. This drive must be the first CD-ROM drive detected and should be connected to your sound card by a small cable, known as a Redbook Audio cable, which enables the game to access the CD audio on the disc. If you follow this instruction, you should be able to successfully launch into the game.
