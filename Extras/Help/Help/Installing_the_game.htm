Installing the Game

1. Before installing, close all other applications. Also, make sure Virtual Memory (located in your System Control Panel under Performance) is not disabled.

2. Insert the game disc into your CD-ROM drive and wait a few moments until the title screen appears. (If the title screen does not appear, please refer to the AutoPlay Troubleshooting section.)

3. Click Install and follow the on-screen instructions.

4. After the game's installation is complete, your computer will install Microsoft's DirectX drivers if you do not already have them. After installation of DirectX is complete, you will need to restart your computer for the new drivers to take effect. For more information on DirectX, see the DirectX section of this file.

5. After rebooting your machine, you can now run the game by choosing Start->Programs->Star Trek: Bridge Commander->Play Star Trek: Bridge Commander  from the Start menu or by clicking Play on the title screen.
